Лабораторно-практическая работа №3 Фабричный метод и абстрактная фабрика (WPF)

Выполнила: Васильева Е.С. Группа: 2407СА1

Сохранить:

Изучить фондовые паттерны проектирования Factory Method и Abstract Factory на примере WPF-приложений.

Задание:

Разработать приложение, создающее фигуры (круг, квадрат, треугольник) цвета цвета. При смене цвета старые фигуры удаляются и принимаются новые.

Реализация показала результат в два этапа:

Фабричный метод

Абстрактная фабрика

Реализация:

Это п 1:

Созданы разработчики классы-создатели для каждого типа фигуры и цвета.

Логика построения вынесена из клиентского кода.

Это п 2:

Создан интерфейс IFigureFactory.

Реализованы фабрики для каждого цвета.

Клиент работает только с абстракцией.

кистей:

Фигуры правильно определены и пересчитаны при смене цвета.

Добавление нового цвета не требует изменения клиентского кода.

Архитектура соответствует принципам слабой связанности и OCP.(Принцип открытости/закрытости)
